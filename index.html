📐 About This Model & Methodology

1

Dataset: 13,000-record synthetic Indian retail credit dataset, RBI/Basel II aligned. Model re-trained using logit-space regression directly on dataset PD values (PD_LR_Model column), achieving 97.55% PD correlation, 99.2% EL correlation, 99.4% Net Profit correlation with the source data. AUC-ROC of classification model: 0.9353. All coefficients, scaler means and standard deviations are exact values from the 13K dataset.

2

Top predictors — DTI Ratio (40.8%) & Loan/Income (21.6%): These two dominate as expected in classical underwriting — the higher the repayment burden and leverage, the greater the default risk. Monthly EMI (15.6%) is the third most influential feature.

3

PD → RAROC pipeline (re-trained on 13K dataset): Model uses logit-space regression on dataset PD values (97.55% corr, 99%+ EL/NP match). EL = PD × LGD × EAD (Basel II). CoF = Loan × CoF% (flat, matching dataset exactly). EC = max(Loan × (√PD × 0.12 + 0.04), EL). RAROC = Net Profit ÷ EC. Decision: PD < 10% → low-risk; RAROC ≥ 15% → profitable.

4

Key insight: The traditional PD-only model (threshold 10%) misses profitability. RAROC extends the framework: HIGH-PD but HIGH-RAROC loans become PRICE PREMIUM (risk compensated by pricing), while LOW-PD but LOW-RAROC loans become REPRICE (under-priced). The 13K dataset has 54.6% APPROVE, 29.3% REJECT, 8.2% PRICE PREMIUM, 7.9% REPRICE.
